# Contributing to Retina Result Fetcher

## How to Contribute

### 1. Reporting Issues
- Use the Bug Report template for technical issues
- Use the Feature Request template for new ideas
- Check Discussions first for existing ideas

### 2. Feature Requests
- Start a Discussion first to get feedback
- Be specific about your use case as a Retina student
- Consider if it's feasible with the Retina API limitations

### 3. Pull Requests
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly on Termux
5. Submit a PR with clear description

### 4. Development Guidelines
- Keep it simple for Termux users
- Maintain backward compatibility
- Add proper error handling
- Update documentation
- Test on actual Android device with Termux
